# Button

## Metadata
- Name: Button
- Category: Controls
- Status: Stable

## Overview
Primary action for booking and secondary action for messaging or calls.
Do not use for navigation within paragraphs.

## Anatomy
- Label
- Optional icon (not currently used)

## Tokens used
- `--color-accent`
- `--color-brand-strong`
- `--color-surface`
- `--radius-pill`
- `--space-*`
- `--transition-button`
- `--elevation-float`

## Props/API
- Variants: `primary`, `ghost`

## States
- Default
- Hover

## Code example
```html
<a class="btn primary" href="#contact">Book Pickup</a>
<a class="btn ghost" href="#contact">WhatsApp Now</a>
```
