# Card

## Metadata
- Name: Card
- Category: Surface
- Status: Stable

## Overview
Generic surface for feature blocks and pricing.
Do not use for full-width sections.

## Anatomy
- Container
- Optional icon
- Title
- Body text

## Tokens used
- `--color-surface`
- `--radius-lg`
- `--elevation-card`
- `--space-*`

## Props/API
None.

## States
- Default

## Code example
```html
<article class="card">
  <div class="icon">👟</div>
  <h3>Shoe Cleaning</h3>
  <p>Deep clean, deodorize, and refresh daily wear and premium shoes.</p>
</article>
```
