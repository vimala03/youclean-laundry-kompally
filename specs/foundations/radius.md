# Radius

## Tokens
- `--ds-radius-sm`: 10px
- `--ds-radius-md`: 14px
- `--ds-radius-lg`: 20px
- `--ds-radius-xl`: 22px
- `--ds-radius-xxl`: 24px
- `--ds-radius-pill`: 999px

## Usage rules
1. Use `--radius-pill` for buttons, chips, tabs.
2. Use `--radius-lg` for cards.
3. Use `--radius-xxl` for hero glass panels.
