# Motion

## Tokens
- `--ds-motion-fast`: 0.2s
- `--ds-motion-slow`: 0.6s
- `--ds-ease-standard`: ease
- `--ds-transition-none`: none

## Aliases
- `--transition-button`
- `--transition-reveal`
- `--transition-none`

## Usage rules
1. Buttons use `--transition-button`.
2. Reveal animations use `--transition-reveal`.
3. Reduced motion uses `--transition-none`.
