## Aliases (Layer 2)
Use `--space-*` tokens directly for padding, margin, gap, and offsets.

## Usage rules
1. No raw pixel values in component CSS.
2. Prefer nearest step up for comfort spacing.
3. Use `--space-0` instead of `0px` for consistency.
