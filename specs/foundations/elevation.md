# Elevation

## Tokens
- `--ds-shadow-card`: 0 20px 60px rgba(0, 54, 72, 0.18)
- `--ds-shadow-float`: 0 10px 30px rgba(0, 209, 124, 0.25)
- `--ds-shadow-soft`: 0 8px 18px rgba(0, 54, 72, 0.12)
- `--ds-shadow-none`: none

## Usage rules
1. Cards use `--elevation-card`.
2. CTAs use `--elevation-float`.
3. Use `--elevation-soft` for subtle accents.
